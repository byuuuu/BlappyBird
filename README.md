# BlappyBird
Blappy Bird! adalah game arkade sederhana yang terinspirasi dari Flappy Bird. Pemain mengendalikan seekor burung dengan cara mengetuk layar agar tetap terbang dan menghindari berbagai rintangan. Semakin jauh pemain bertahan, semakin tinggi skor yang diperoleh.
